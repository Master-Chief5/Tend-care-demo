# Working agreements

## Communication
- When a task is likely to take **more than ~5 minutes**, give a time estimate
  up front (before starting the work). Short/trivial tasks don't need one.

## Deploy / URLs (Vercel)
- The live site is the canonical domain **`tend-care-demo.vercel.app`**.
- Per-deploy and preview URLs (the long `…-master-chief5s-projects.vercel.app`
  / `…-git-<branch>-…vercel.app` forms) are pinned to a single old build and
  never update — don't point the user at those for "is it live yet?" checks.
